# writers-directory-c
Console application for storing and searching information about writers.
## О проекте
Электронный справочник писателей, разработанный в рамках учебного проекта.
Программа позволяет хранить, искать и изменять информацию о писателях.
## Description
This project is an electronic reference system about writers.
The application allows users to store, search and manage information about authors.
## Features
- show a splash screen with pseudographics.
- see all the writers.
- add a new author and his biography.
- find the right writer.
- saving information to the file.
- working with the database.
- view information about the author.
## Project Structure
- main.c
- writers.txt
## Technologies
- C
- File I/O
- Console application
- Structures
- Arrays
## How to Run
1. Open Visual Studio 
2. Build the project
